Данный репозиторий содержит все практические работы 2 семестра по дисциплине «Фронтенд и бэкенд разработка».

## Организация репозитория

Согласно требованиям заданий, для каждой практической работы предполагалось создание отдельного репозитория.

В целях упрощения структуры и исключения дублирования репозиториев все работы размещены в рамках одного репозитория, но распределены по отдельным веткам.

Каждая ветка соответствует конкретной практической работе или группе практических работ.

### Ветки:

- Ветка `FB-4-PR-1-2` — практические работы №1–2  
  https://github.com/CatherineFlower/Front_and_Back_Sem_2/tree/FB-4-PR-1-2
- Ветка `PR-3-4` - практические работы №3-4
  https://github.com/CatherineFlower/Front_and_Back_Sem_2/tree/PR-3-4

Аналогичным образом остальные практические работы будут размещены в соответствующих ветках.

## Принцип распределения

- Каждая практическая работа выполнена в отдельной ветке.
- Ветка содержит завершённую версию соответствующей практики.
- История коммитов отражает процесс разработки.

## Проверка работ

Для просмотра конкретной практической работы необходимо:

1. Перейти во вкладку **Branches**.
2. Выбрать соответствующую ветку.
3. Ознакомиться с содержимым проекта данной практики.
