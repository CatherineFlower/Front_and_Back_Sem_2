# Практические работы 3–4 по дисциплине «Фронтенд и бэкенд разработка»

## Информация о работе

В данной ветке представлено решение практических работ №3 и №4 по дисциплине «Фронтенд и бэкенд разработка».

Практическая работа №3 посвящена изучению JSON, тестированию собственного API и выполнению запросов через Postman.  
Практическая работа №4 посвящена подключению React-приложения к backend на Express и реализации CRUD-операций через пользовательский интерфейс.

## Практическая работа №3

В рамках данной практической работы был запущен backend на Express и протестирован через Postman.

### Что реализовано

- запуск Express API;
- работа с JSON-запросами и JSON-ответами;
- получение списка товаров;
- получение товара по идентификатору;
- создание нового товара;
- частичное обновление товара;
- удаление товара;
- валидация входных данных;
- обработка ошибок 400 и 404;
- подготовка проекта для проверки запросов через Postman.

### Запуск backend

```bash
cd backend
npm install
npm run dev
```

После запуска сервер доступен по адресу:

```text
http://localhost:3000
```

### Примеры запросов

Получить список товаров:

```http
GET /api/products
```

Получить товар по id:

```http
GET /api/products/:id
```

Создать товар:

```http
POST /api/products
Content-Type: application/json
```

Пример JSON:

```json
{
  "title": "Ноутбук",
  "category": "Электроника",
  "description": "Лёгкий ноутбук для учёбы и работы",
  "price": 59990,
  "stock": 7
}
```

Изменить товар:

```http
PATCH /api/products/:id
Content-Type: application/json
```

Пример JSON:

```json
{
  "price": 60990,
  "stock": 5
}
```

Удалить товар:

```http
DELETE /api/products/:id
```

## Практическая работа №4

В рамках данной практической работы было разработано React-приложение, подключённое к Express API.

### Что реализовано

- подключение frontend к backend;
- получение списка товаров с сервера;
- добавление нового товара через форму;
- редактирование товара;
- удаление товара;
- обновление цены товара;
- обработка ошибок на клиенте;
- отображение данных из API в интерфейсе.

### Запуск frontend

```bash
cd frontend
npm install
npm run dev
```

После запуска приложение доступно по адресу:

```text
http://localhost:3001
```
