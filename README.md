# Практические работы 1–2 по дисциплине «Фронтенд и бэкенд разработка»

## Информация о работе

В данной ветке представлено решение практических работ №1 и №2 по дисциплине «Фронтенд и бэкенд разработка».

Практическая работа №1 посвящена созданию карточки товара с использованием HTML и SCSS.  
Практическая работа №2 посвящена разработке простого backend-приложения на Node.js и Express с реализацией CRUD-операций для сущности «товар».

## Практическая работа №1

В рамках первой практической работы была разработана адаптивная карточка товара.

При выполнении задания были использованы следующие возможности SCSS:

- переменные для хранения основных цветов, отступов и размеров;
- миксины для повторно используемых стилей;
- вложенность селекторов;
- адаптивное оформление для корректного отображения на разных размерах экрана.

### Что реализовано

- карточка товара с названием, описанием, характеристиками и ценой;
- кнопка действия;
- современное визуальное оформление;
- адаптивная верстка;
- стилизация через SCSS.

### Запуск проекта

Перейти в папку `frontend`:

```bash
cd frontend
```

Установить зависимости:

```bash
npm install
```

Собрать SCSS в CSS:

```bash
npm run build:sass
```

После этого открыть файл `index.html` в браузере через встроенный сервер IDE.

## Практическая работа №2

В рамках второй практической работы было реализовано серверное приложение на Express с CRUD-операциями для товаров.

### Используемые технологии

- Node.js
- Express
- CORS
- Nodemon

### Что реализовано

- запуск HTTP-сервера;
- обработка JSON-запросов;
- логирование входящих запросов;
- получение списка товаров;
- получение товара по идентификатору;
- создание нового товара;
- частичное обновление товара;
- удаление товара;
- базовая валидация входных данных;
- возврат корректных HTTP-статусов и сообщений об ошибках.

### Запуск backend

Перейти в папку `backend`:

```bash
cd backend
```

Установить зависимости:

```bash
npm install
```

Запустить сервер в режиме разработки:

```bash
npm run dev
```

После запуска сервер будет доступен по адресу:

```text
http://localhost:3000
```

## API

### Получить список товаров

```http
GET /api/products
```

### Получить товар по id

```http
GET /api/products/:id
```

### Создать новый товар

```http
POST /api/products
Content-Type: application/json
```

Пример тела запроса:

```json
{
  "title": "Новый товар",
  "price": 1990
}
```

### Обновить товар

```http
PATCH /api/products/:id
Content-Type: application/json
```

Пример тела запроса:

```json
{
  "price": 2490
}
```

### Удалить товар

```http
DELETE /api/products/:id
```

## Результат

В данной ветке содержится завершенное решение практических работ №1 и №2.  
Frontend-часть включает оформленную карточку товара, выполненную с использованием SCSS.  
Backend-часть реализует CRUD API для работы с товарами и готова к проверке.
